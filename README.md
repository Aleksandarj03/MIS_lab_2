# lab_2

Laboratoriska vezba 2 - Recipe App with Firebase Integration

## Features

### 1. Favorite Recipes
- Add recipes to favorites by tapping the heart icon on meal cards
- View all favorite recipes in a dedicated favorites screen
- Favorites are stored locally using SharedPreferences
- Access favorites from the home screen via the heart icon in the app bar

### 2. Firebase Integration
- Firebase Core integration for backend services
- Firebase Cloud Messaging for push notifications
- See `FIREBASE_SETUP.md` for detailed setup instructions

### 3. Daily Recipe Notifications
- Receive daily notifications with a random recipe suggestion
- Configurable notification time
- Test mode available for testing (sends notification every 10 seconds)
- Access notification settings from the home screen via the bell icon

## Setup Instructions

### 1. Install Dependencies

```bash
flutter pub get
```

### 2. Firebase Setup

**Important:** You must set up Firebase before running the app.

1. Follow the instructions in `FIREBASE_SETUP.md`
2. Run `flutterfire configure` to generate Firebase configuration
3. Place `google-services.json` in `android/app/` directory

### 3. Run the App

```bash
flutter run
```

## Usage

### Adding Favorites
1. Browse categories or search for recipes
2. Tap the heart icon on any meal card to add it to favorites
3. Access favorites from the heart icon in the app bar

### Configuring Notifications
1. Tap the bell icon in the app bar
2. Enable/disable notifications
3. Set notification time (or enable test mode for testing)
4. Test mode sends notifications every 10 seconds

## Testing Notifications

1. Open the app
2. Go to Notification Settings (bell icon)
3. Enable "Test Mode" to receive notifications every 10 seconds
4. Disable "Test Mode" and set a specific time for daily notifications

## Dependencies

- `firebase_core`: Firebase Core SDK
- `firebase_messaging`: Firebase Cloud Messaging
- `flutter_local_notifications`: Local notifications
- `shared_preferences`: Local storage for favorites
- `http`: HTTP requests to TheMealDB API
- `url_launcher`: Open YouTube links
